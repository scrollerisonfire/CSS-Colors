This project is a simple HTML file that demonstrates basic usage of colors in CSS.

## What it includes
- Basic HTML structure with `h1` and `h2` elements
- Embedded CSS using the `<style>` tag
- Usage of:
  - `background-color` for the page background
  - `color` for text color
  - Named colors and hex color values

## Purpose
This project is created as a practice exercise for learning:
- CSS selectors (`html`, `h1`, `h2`)
- The difference between text color and background color
- Basic CSS styling principles

Suitable for beginners learning HTML and CSS.
